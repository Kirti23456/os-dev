      ##My operting system project(part 1)
This project is a basic x86 operating system kernel built from scratch using C.

      ##IN THIS PART WE ARE HAVING FOLLOWING THINGS
      
*custom freestanding kernel written in C
*Boots using GRUB(MULTIBOOT COMPLIANT)
*Directly writes output to vga text buffer
*tested using qemu emulator

      ##what this os does
*loads via GRUB bootloader 
*executes a customer kernel entry point 
*displays a character on screen using VGA memory

      ##TECHNOLOGUES USED
*c (freestanding , no standard library)
*GNU Linker (id)
*GRUB Bootloader 
-QEMU Emulator
-Linux(Ubantu)
